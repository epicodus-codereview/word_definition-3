Word & Definition

By: Ryan Heupler

Description:

Lets a user type in a word that will save to a list. A user can then go in and
add a definition to that word.

Installation

-Copy clone link
-Type 'Git clone {link}' in terminal
-Run 'Ruby app.rb'

Technologies:

Ruby, Sinatra, CSS, HTML
